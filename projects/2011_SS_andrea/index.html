Von Blender zur BeuthGameEngine

Die BeuthGameEngine bietet die Möglichkeit, Modelle aus Blender über eine Collada-Schnittstelle zu importieren.
Dafür müssen einige Voraussetzungen erfüllt sein.


1. Es muss die Version 2.5 von Blender verwendet werden 2. Mesh hinzufügen 3. Für die BeuthGameEngine müssen Vierecke zu Dreiecken umgewandelt werden.
1. Alle Vertices markieren
2. Edit-Modus, Mesh/Faces/QuadsToTris 4. Collada exportieren 5. Bitte die exportierte Collada-Datei überprüfen. Im Tag polylist/vcount darf nur der Wert 3 vorkommen. Stehen dort Vieren, dann hat das die Methode QuadsToTris nicht funktioniert. Bitte 3. wiederholen. 6. Das Tool zum Konverieren aufrufen. 7. Das Modell zur Scene hinzufügen 8. Freuen 9. Farbige Modelle über den Menüpunkt Materials in Blender 10. Das farbige Modell im Web 11. Momentan werden Szenen mit mehreren Modellen noch nicht unterstützt. Man kann aber in Blender mehrere Objekte über Join zusammenführen 2. Das zusammengführte Modell im Web jQuery Popup Window by VisualLightBox.com v4.8
